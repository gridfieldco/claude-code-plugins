The user has pasted an error message or stack trace. Do the following:

1. **What happened**: Explain the error in plain language (1-2 sentences)
2. **Root cause**: Identify the most likely cause based on the error type and context
3. **Fix**: Provide the most direct fix, with a code snippet if applicable
4. **Prevention**: One sentence on how to avoid this in the future

If the error references specific files in the project, read them to give more targeted advice. Keep the total response under 20 lines.
